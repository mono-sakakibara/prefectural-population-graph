## 導入ツール

### フレームワーク

使ったことのある Next.js で実装を行おうと思いましたが、新しい技術を使うチャンスだと思い、preact + vite の構成で挑戦してみました。

#### preact

- 理由
  今回は１ページのみのシンプルな SPA の開発のため、容量は小さいに越したことはないと思い、今回の実装に適していると考えました。

#### vite

- 理由
  高速ということで有名な点、現在の実務でも使う可能性があることから、vite を選択しました。
